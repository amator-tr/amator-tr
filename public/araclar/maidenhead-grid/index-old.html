Maidenhead Grid Hesaplayıcı

Amatör telsiz grid locator (IARU) ↔ enlem/boylam dönüşümü, mesafe ve bearing.

Maidenhead nedir? Dünyayı 18×18×10×10×24×24 hücreye bölen, 6-8 karakterli amatör radyo koordinat sistemi. Türkiye KM-KN field'larında. Örn: İstanbul ≈ KN41la, Ankara ≈ KM69kw, İzmir ≈ KM38nk. DX, FT8, contest ve POTA için gereklidir. DXCC tutorial →

Grid → Lat/Lon

İki Grid Arası Mesafe + Bearing

Format açıklaması: KN41 = Field (KN, 20°×10°) + Square (41, 2°×1°), ~150 km hassasiyet. KN41la = + Subsquare (lj, 5'×2.5'), ~5 km hassasiyet (FT8/contest standart). KN41la11 = + Extended (8-char), ~250 m hassasiyet (DXcluster spotting).
